## 🌟 Project

### ☕ AI Kiosk for Cafe (Korean)

📅 **Project Duration**: 2025.03.17 ~ 2025.04.06  
🔗 **Repository**: [Korean_AI_Kiosk_Agent](https://github.com/Wanted-AI-project/Korean_AI_Kiosk_Agent)

#### 💡 Project Overview

AI 키오스크 프로젝트는 카페 환경에서 음성 기반 자연어 인터페이스를 통해 고객이 주문을 보다 직관적으로 진행할 수 있도록 돕는 **음성 주문 챗봇 시스템**입니다.
고객은 대화하듯 메뉴를 문의하거나 영양 정보를 확인하고, 원하는 메뉴를 장바구니에 담아 최종 주문까지 자연스럽게 이어갈 수 있습니다.

### 🎮 AI를 이용한 게임 내 음성명령 시스템

📅 **Project Duration**: 2025.03.13 ~ 2025.03.14  
🔗 **Repository**: [VocieCommand]([https://github.com/Wanted-AI-project/Korean_AI_Kiosk_Agent](https://github.com/0-9-10-12/VoiceCommand))

#### 💡 Project Overview

게임 내 몬스터에게 음성 명령을 전달하면, 그 명령을 수행하게 하는 **음성 명령 변환 시스템**입니다.

사용자의 음성 명령을 받아 자연어 텍스트로 전환 -> 텍스트를 JSON 형태로 파싱하도록 LLM모델을 튜닝 -> RestAPI 서버로 전달
